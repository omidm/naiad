First Naiad Program, it simply reads an input from terminal and prints in back.
in the distributed mode, it reads the input from all process and prints them
out over the main process terminal.

To compile:
    $ make

To run over single node:
    $ make run

To run over two nodes on your local machine:
    $ make run-dist
